# curve_fitting
C++ implementation of curve fitting

## requirement
- Eigen (>= 3)

## build and install
```
$ git clone https://github.com/amslabtech/curve_fitting.git
$ cd curve_fitting
$ mkdir build && cd build
$ cmake ..
$ make
$ sudo make install
# for uninstall
$ cat install_manifest.txt | xargs sudo rm -rf
```
